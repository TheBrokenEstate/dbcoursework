About StudentTrack

StudentTrack is a program designed to aid teachers in organising and managing their courses, students and classes. Please read the "Setup and Installation" section to get started.

Getting Started

To get started on StudentTrack it is recommended that you first go to the Courses page and add a course. After this you can add topics to this course, and add and assign classes to take this course. Once you have this initial setup done you can add some upcoming homeworks and tests so we can remind you when their scheduled time approaches. StudentTrack can also track the performance of classes over time, if you record your students' progress in StudentTrack you can view statistics on recent performance, monitor completion of homeworks and tests, and get general reminders on the Dashboard.

Setup and Installation

For Testing and Assessment
  1. Ensure python 3.6 or greater is installed.
  2. Open CMD on Windows, or a terminal on Linux.
  3. Navigate to the prototypes directory. (By default this document is contained in it.)
  4. If you wish to use a virtual enviroment run `python -m venv ./venv` and then `./venv/Scripts/activate.bat` on Windows, or `./venv/Scripts/activate` on Linux.
  5. Run `pip install -r requirements.txt` to install the requirements for this program.
  6. Run `python setup-database.py` to initialize the database.
  7. Run `python run.py` to run the server.
  8. Open a browser and navagate to `localhost:5000`.
